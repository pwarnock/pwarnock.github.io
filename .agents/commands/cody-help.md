# Cody Help Command

Show available Cody PBT commands.

## Available Commands
- :cody help - Show all commands
- :cody plan - Start planning (discovery, PRD, plan)
- :cody build - Create feature backlog
- :cody version build - Create specific version
- :cody version add - Add version to backlog
- :cody refresh - Refresh memory
- :cody upgrade - Upgrade framework
- :cody assets list - List assets
